# Employee Management System

A simple **menu-driven Employee Management System built using Python**.
This project was developed as part of my **Python Full Stack Development
training** to practice Python fundamentals and modular programming.

## 🚀 Features

-   Add a new employee
-   View all employees
-   Search an employee by Employee ID
-   Edit employee details
-   Delete an employee
-   Exit the application
-   Menu-driven command-line interface

## 📄 Module Description

  File                   Description
  ---------------------- --------------------------------------------
  `app.py`               Main application and menu-driven program
  `data.py`              Stores employee data in a list
  `employee_data.py`     Handles adding employee details
  `employee_view.py`     Displays employee records
  `employee_search.py`   Searches for an employee using Employee ID
  `employee_edit.py`     Updates employee name and salary
  `employee_delete.py`   Deletes an employee record


## 🔄 How It Works

1.  The application displays the Employee Management System menu.
2.  The user selects an option from `1` to `6`.
3.  The main program calls the corresponding employee module.
4.  Employee information is maintained using Python data structures.
5.  The menu continues until the user selects the Exit option.
